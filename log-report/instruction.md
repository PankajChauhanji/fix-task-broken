There is an access log (`access.log`) in the working directory. Analyze the traffic and summarize what you find. 
Save your findings to a file named `report.json` in the working directory. The file must be a valid JSON object satisfying these numbered success criteria:

1. `total_requests`: The total number of requests in the log file.
2. `unique_ips`: The total number of unique client IP addresses involved.
3. `top_path`: The single most frequently requested path.
